### Development

You'll need [cargo](https://doc.rust-lang.org/cargo/) installed on your machine
to spawn the various components the project needs to run locally.

First, set the environment variables:

```
DISCORD_BOT_TOKEN=
OPENAI_API_KEY=
```

Run the bot:

```
cargo run
```
